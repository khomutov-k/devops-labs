## Using Helm
1. Encrypting creds in secrets1.yaml

2. Adding secret.yaml to templates

```kubernetes helm
apiVersion: v1
kind: Secret
metadata:
  name: user-pass
  namespace: default
  labels:
    app: java-app
type: Opaque
data:
  password1: {{ .Values.password1 | b64enc | quote }}
  username1: {{ .Values.username1 | b64enc | quote }}
```
